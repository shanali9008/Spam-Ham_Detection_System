# Spam-Ham_Detection_System
## Overview:
<br>
This project is a Machine Learning-based system that classifies text messages as Spam or Ham (Not Spam).
<br>
Although spam detection is a well-established problem, this project was developed to strengthen my understanding of `NLP` and `ML` workflows and to experiment with improving model performance through preprocessing and model tuning.
<br>

## Objectives:
* Understand the end-to-end pipeline of a text classification system
* Apply Natural Language Processing (NLP) techniques
* Compare different machine learning models
* Improve performance through feature engineering and tuning
<br>

## Tools/Libraries:
* python
* jupyter notebook 
* streamlit
* pandas, nltk, numpy, matplotlib, seaborn, sklearn
* NLP techinques (Tokenization, Vectorization)
<b>

* SVM
